Datos para Recibir Transferencias Internacionales

Chile

Nombre del Banco: Ejemplo: Banco de Chile
Tipo de Cuenta: Corriente o Vista (RUT)
Número de Cuenta: Ejemplo: 1234567890
RUT: Ejemplo: 12.345.678-9
Nombre del Titular: Nombre Apellido
Email: ejemplo@email.com
Código SWIFT/BIC: Ejemplo: BCHICLRM

Nota: Las transferencias nacionales usan el RUT y número de cuenta.

Colombia

Nombre del Banco: Ejemplo: Bancolombia
Tipo de Cuenta: Ahorros o Corriente
Número de Cuenta: Ejemplo: 12345678901
Cédula: Ejemplo: 1234567890
Nombre del Titular: Nombre Apellido
Código SWIFT/BIC: Ejemplo: COLOCOBM

Nota: Para transferencias nacionales solo se necesita el número de cuenta.

Perú

Nombre del Banco: Ejemplo: BCP
Tipo de Cuenta: Ahorros o Corriente
Número de Cuenta: Ejemplo: 1234567890123456
DNI: Ejemplo: 12345678
Nombre del Titular: Nombre Apellido
Código SWIFT/BIC: Ejemplo: BCPLPEPL
Código Interbancario (CCI): Ejemplo: 0021231234567890123456

Nota: Para transferencias nacionales se usa el número de cuenta. El CCI se usa para transferencias interbancarias.

Ecuador

Nombre del Banco: Ejemplo: Banco Pichincha
Tipo de Cuenta: Ahorros o Corriente
Número de Cuenta: Ejemplo: 1234567890
Cédula/RUC: Ejemplo: 1234567890
Nombre del Titular: Nombre Apellido
Código SWIFT/BIC: Ejemplo: PICHECEQ

Nota: Ecuador usa el dólar estadounidense como moneda oficial.

Venezuela

Nombre del Banco: Ejemplo: Banco de Venezuela
Tipo de Cuenta: Ahorros o Corriente
Número de Cuenta: Ejemplo: 0102033445566778899
Cédula: Ejemplo: V-12345678
Nombre del Titular: Nombre Apellido
Código SWIFT/BIC: Ejemplo: BDVZVECA

Nota: Debido a regulaciones, las transferencias internacionales pueden estar sujetas a restricciones. Verificar con el banco.

España

Nombre del Banco: Ejemplo: Banco Santander
IBAN: Ejemplo: ES91 2100 0418 4502 0005 1332
Número de Cuenta: Ejemplo: 2100 0418 4502 0005 1332
DNI/NIE: Ejemplo: 12345678A
Nombre del Titular: Nombre Apellido
Código SWIFT/BIC: Ejemplo: BSCHESMM

Nota: Para transferencias dentro de la UE se puede usar solo el IBAN.

Estados Unidos

Nombre del Banco: Ejemplo: Bank of America
Número de Ruta (ABA): Ejemplo: 026009593
Número de Cuenta: Ejemplo: 123456789012
Tipo de Cuenta: Checking (corriente) o Savings (ahorros)
Nombre del Titular: Nombre Apellido
Dirección del Banco: Ejemplo: 100 N Tryon St, Charlotte, NC 28202
Código SWIFT/BIC: Ejemplo: BOFAUS3N

Nota: Para transferencias nacionales (dentro de EE.UU.) se necesita el número de ruta ABA y número de cuenta.
